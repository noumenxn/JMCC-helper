# JMCC Compiler
VSCode расширение для [JustMC compiler](https://github.com/donzgold/JustMC_compilator)

## Возможности
### 1. **Подсветка синтаксиса**
Подсветка синтаксиса для файлов `.jc`.

### 2. **Автодополнение**
Контекстно-зависимое автодополнение для объектов justcode.

### 3. **Информация при наведении**
Подробная информация об justcode объектах при наведении курсора.

### 4. **Сигнатуры функций**
Отображает параметры во время ввода.

### 5. **Компиляция и декомпиляция**
Компиляция `.jc` файлов в:
  - **URL** для загрузки на сервер.
  - **Локальные JSON файлы**
Декомпиляция `.json` файлов обратно в формат JustCode.
Кнопка `F5` соответственно (де)компилирует открытый json/jc файл

## **Настройки**
Все настройка расширения устанавливаются в меню `Settings`.
Имеет следующие параметры:
- `Clear Terminal` - очищать терминал перед выполнением команды
- `Compiler Output Path` - путь для сохранения результата компиляции 
- `Compiler Path` - путь до компилятора jmcc.py (оставьте пустым для встроенного)
- `Default Compile Mode` - режим компиляции по кнопке F5 и Play
- `Hide Completion` - не отображать автодополнения
- `Hide Hover` - не отображать информацию при наведении
- `Hide Signature Help` - не отображать сигнатуры
- `Obfuscate_code` - обфусцировать выдаваемый код
- `Obfuscator_key` - ключ обфускации, по умолчанию имя компьютера (не обфусцирует сохраненные переменные)
- `Check_updates` - проверять обновления
- `Auto_update` - обновлять компилятор
- `Check_beta_versions` - учитывать бета-версии при обновлениях
- `Code_line_limit` - длина строки в выдаваемом коде
- `Create_function_descriptions` - создавать описание функциям
- `Create_world_restart_function` - создавать функцию перезагрузки мира
- `Default_variable_type` - тип для необъявленных переменных
- `Lang` - язык сообщений компилятора
- `Optimize_code` - экспериментальная оптимизация
